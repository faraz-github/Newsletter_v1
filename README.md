# Newsletter_v1

A simple Newsletter subscription app made using Nodejs environment using Mailchimp API.

## Available Commands

- npm install
- node app.js

## Installation

Newsletter_v1 requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
cd Newsletter_v1
npm i
node app.js
```
